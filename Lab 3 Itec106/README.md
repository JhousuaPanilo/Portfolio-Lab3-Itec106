# Professional Portfolio Website

A responsive static portfolio website built with HTML, CSS, and JavaScript.

## Files

- `index.html` – main page markup
- `styles.css` – responsive styling and visual design
- `script.js` – mobile navigation behavior

## How to use

1. Open `index.html` in your browser.
2. Replace placeholder content with your own name, bio, projects, and contact info.
3. Update the image URL in `index.html` to your own profile photo.

## Deployment

You can host this site using any static hosting service such as GitHub Pages, Netlify, or Vercel.
